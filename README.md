🎬 TikTok (Flutter Reels Clone)

A short-video social media app built with Flutter that replicates TikTok-style vertical video feeds with smooth performance, caching, and auto-play behavior.

🚀 Features
🔥 Fetches video data (URLs, captions, likes, username) from Firebase Firestore
⚡ Preloads next 3–4 videos for seamless scrolling experience
💾 Caches videos locally to avoid re-downloading and improve performance
🎯 Auto play/pause videos based on visibility in scroll
❤️ Like system with real-time Firestore updates
🧠 Smooth UI overlay showing username, caption, and like count
📱 TikTok-style full-screen vertical video feed
🎥 Optimized video playback using cached video player
🛠️ Tech Stack
Flutter 💙
Firebase Firestore 🔥
Video Player (cached_video_player_plus)
State Management (Stateful widgets)
Dart
📌 Highlights

This project focuses on:

Smooth video streaming experience
Performance optimization using caching & preloading
Social media-style UI/UX
Scalable Firestore structure for reels
